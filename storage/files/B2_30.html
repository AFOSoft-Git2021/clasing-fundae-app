Using a Computer

Address book

A database that stores names, addresses and other contact information for a computer user.

  • She copied all the addresses into her address book.
  • She copied the phone number into her address book.
  • I flipped through my address book but couldn't find her phone number.

Browser

A computer program that makes it possible for you to read information on the internet:

  • The latest version of the browser allows you to listen to the radio while you surf the net.
  • Web designers have to make sure their pages are compatible with most browsers.
  • Clicking on this link will open a new browser window.
  • You'll need a browser to navigate the Web.
  • Use the browser to view photographs downloaded from your digital camera.

Email

Messages sent electronically over a computer network

  • It's important to speak to people directly from time to time, rather than always using email.
  • You can contact us through our website, by phone, or by email.
  • I'll forward his email to you if you're interested.
  • I'll send you my email address once I'm online.
  • How did people communicate in the days before email?

Download

A computer program or information that has been or can be copied into a computer's memory

  • The file will only take a few seconds to download.
  • The user confirms their order and downloads an applet which asks them for their personal and payment details.
  • Software updates can be downloaded from the company's website.

Antivirus software

a program that finds and removes viruses from a computer

  • You need to update your antivirus software regularly.
  • Antivirus software is a simple safeguard that many computer users have not bothered to install.
  • Setting up and running preventative maintenance programmes and anti-virus software updates.

Backup

a copy of information held on a computer that is stored separately from the computer.

  • Before we leave work each day, we make a backup of all the records we have entered into the computer that day.
  • The department's backup disks are all stored in a different building.

Bug

a (small) defect or fault in a program

  • A bug caused the company's computer system to crash.
  • If you get a bug in your system, please report it as soon as possible so it can be fixed.
  • This program keeps crashing - there must be a bug somewhere.
  • The software producers are currently trying to fix two significant bugs.
  • This game keeps freezing - am I doing something wrong or do you think it's a bug of some sort?
  • They've produced a downloadable patch that will repair the bug in the program.

Cache

 a type of computer memory for storing information temporarily and getting to it quickly

  • All browsers create temporary files that speed up your internet surfing by caching parts of web pages that you access.
  • The story is no longer posted on the website but is cached online by search engines.
  • The tool lets you specify what types of data you want cached.
  • It downloads code over the network, caches it, and runs it in a secure environment.

Font

a particular sort of lettering (on the screen or on paper). Arial is a font. Times New Roman is another.

  • When you download the font it may be in a compressed format, such as a zip file.
  •  If you double click on one of the font names a sample is displayed on the screen.
  • Object properties can include font, style type, fill or color palette.
  • This information base is then manipulated using a font descriptor file to generate a particular font.

Format

All hard disks and floppy disks have to be electronically prepared for use by a process called formatting. Hard disks are pre-formatted by the computer manufacturer. If you buy a floppy disk that is not pre-formatted, you format it yourself, using a program that comes with your PC.

  • Please format this floppy disc.
  • The new format has unanimous support and could be introduced next season.
  • When you download the font it may be in a compressed format, such as a zip file.

File

 a specific computer record. It could contain data such as text (e.g. essay.doc), or a program such as paint.exe.

  • I’m going to copy/save this file.
  • I will send you a copy of the file.
  • Make a file for these documents and write 'finance' on the tab.

Hard disk

A device that is fixed inside a computer and is used to store programs and information.

  • Hard disk drive technology continues its inevitable march toward greater areal densities, and hybrid drives are being purchased in greater numbers.
  • I found that the hard disk had been wiped.
  • That information gets downloaded onto a hard disk.

Peripheral

any equipment that is connected externally to a computer. For example, printers, scanners and modems are peripherals.

  • The book contains a great deal of peripheral detail.
  • The system is expected to boost sales of PCs as well as components, applications, and peripheral devices.

Pixel

the image that you see on the screen is made of thousands of tiny dots, points or pixels

  • A high-definition television uses about two million pixels per screen.
  • Each pixel in a plasma has only two states: on or off.
  • I know exactly where everything is on the screen, down to the pixel.
